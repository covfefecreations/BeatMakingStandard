1. Foundation — Set Your Canvas

Start a new project in BandLab. Set tempo, key, and choose a drum kit.

  • New Project → Create from Scratch
  • Set BPM (tempo) & choose key
  • Add Drum Machine / Drum Track (e.g., 808 or Lo-Fi)
  • Save project with clear name + tags
Anchor phrase: “Set the stage” Icon motif: slider → canvas
Notes: Quick presets save time. For teaching, include sample BPM presets for genres (Hip-Hop 70–95, Pop 100–130, EDM 120–150).

2. Rhythm — Build the Groove

Create a strong 4-bar loop focusing on kick + snare, then hats and percussion for texture.

  • Program kick + snare first
  • Add hi-hat pattern (vary velocities)
  • Layer percussion & fills
  • Loop and pick the strongest 4 bars
Technique: humanize velocity Teaching tip: show before/after dynamics
Notes: Use simple call-and-response between kick and snare; use offbeat hats to add forward motion. Keep low complexity early.

3. Melody — Create the Hook

Add instrument tracks for chords & a short repeating motif that becomes your hook.

  • Add Software Instrument (synth/piano/pad)
  • Create chord progression (simple templates: I–VI–V–IV)
  • Write short melodic motif (4 bars)
  • Quantize lightly — keep musical timing
Anchor phrase: “Hook > Hype” Icon motif: keys → motif
Notes: Use inversion and sparse voicings to make space for the vocal. Duplicate motif across octaves for texture.

4. Bass — Ground the Energy

Add 808 or bass synth tuned to the key; lock it rhythmically with the kick.

  • Choose bass instrument (808, sub, synth bass)
  • Tune bass notes to the project key
  • Align bass hits with kicks for cohesion
  • Gentle sidechain or ducking if needed
Technique: tune 808 to note Icon motif: low waves
Notes: If using an 808, make sure the tail doesn't clash with kick; use a transient shaper or short fade for clarity.
